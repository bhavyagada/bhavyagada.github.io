Trained a Convolutional Neural Network on handwritten numerical digits, achieving more than 95% accuracy.

Users hold an unsolved sudoku in front of their webcam. The program recognizes the digits, creates the sudoku grid, solves the puzzle, and writes the missing cell digits back to the screen within the blink of an eye.

The code is designed to be easily extendable for use in solving various puzzles, including but not limited to N-Puzzle, KenKen, Crosswords and Scrabble.
